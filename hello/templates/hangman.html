Hangman

This is the game Hangman.

Here is how you play:

· The game will tell you the number of guesses you have left, the leters you have left to guess , the number of warnings you have left, the word with underscores as the letters you havn't guessed yet.

· It should then ask you for a letter.

· If you put in a letter, then it will tell you weather it is right or wrong.

· If it is right, then you will get that letter in the word.

· If it is wrong, then you will lose a guess. If you have no guesses left, then you will lose a warning.

· If you put in a character that is not a letter, you will lose a warning.
